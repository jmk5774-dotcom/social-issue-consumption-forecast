# Social Issue-Based Consumption Forecasting

## 프로젝트 개요

사회 이슈가 소비 패턴에 미치는 영향을 정량적으로 분석하고 미래 소비량을 예측하기 위해 수행한 시계열 예측 프로젝트입니다.

기존 시계열 모델은 과거 데이터만 활용하기 때문에 사회적 이슈나 외부 환경 변화가 발생할 경우 예측 성능이 저하되는 한계가 있습니다. 이를 개선하기 위해 뉴스 데이터를 활용하여 사회 이슈 관련 외생변수를 생성하고 Prophet 모델에 반영하여 예측 성능을 향상시키고자 하였습니다.

---

## 프로젝트 기간

* 2024.03 ~ 2024.11

* 소비 패턴 시계열 예측

* 뉴스 기반 외생변수 생성
* Prophet 모델 성능 개선

## 데이터 구성

### 종속 변수

* 소비 패턴 데이터

### 외생 변수

뉴스 데이터를 전처리하여 다음 정보를 변수로 생성

* 사회 이슈 발생 빈도
* 이해관계자 개입 여부
* 뉴스 키워드 기반 영향도 지표

총 14개의 외생변수(C1~C7, M1~M7)를 생성하여 모델에 적용

---

## 프로젝트 수행 과정

### 1. 데이터 전처리

* 시계열 데이터 정제
* 날짜 형식 통일
* Prophet 입력 형식(ds, y) 변환

### 2. 외생변수 설계

* 뉴스 데이터 수집 및 전처리
* 사회 이슈 발생 빈도 분석
* 이해관계자 개입 여부 변수화
* 외생변수(C1~C7, M1~M7) 생성

### 3. Prophet 모델 구축

* Prophet 기반 시계열 예측 모델 학습
* 외생변수 추가
* Parameter Grid Search 기반 하이퍼파라미터 튜닝

### 4. 성능 평가

평가 지표

* RMSE
* MAPE

예측값과 실제값 비교를 통해 모델 성능을 검증

## 주요 성과

* 사회 이슈 데이터를 활용한 외생변수 설계
* Prophet 기반 시계열 예측 모델 구축
* 뉴스 데이터를 활용한 소비 패턴 변화 분석
* 기존 오차율 0.145 → 0.063 개선
* 약 56% 수준의 예측 성능 향상 달성

## 배운 점

본 프로젝트를 통해 시계열 예측에서는 단순히 과거 데이터만 활용하는 것이 아니라, 도메인 지식을 기반으로 외부 요인을 변수화하는 과정이 중요하다는 점을 배웠습니다. 또한 데이터 분석에서 모델 선택뿐 아니라 문제 정의와 변수 설계가 예측 성능에 큰 영향을 미친다는 사실을 경험할 수 있었습니다.
